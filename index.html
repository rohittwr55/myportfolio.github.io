Work and Technology!

In my previous project, I utilized Tableau for Data Visualization. The typical process we followed involved initially receiving the INCIDENT via ServiceNow from the client, along with their full requirements. Often, the client would provide their requirements through an Excel sheet, which we then used for visualization purposes, or we retrieved the data from the MySQL server.

Once we received the client's requirements, we commenced work on the incident. Upon obtaining the data, whether in CSV, Excel, or MySQL format, we proceeded with data cleaning using Tableau Prep.

Following data cleaning, we addressed any missing data or uncertainties by communicating with the user and implementing necessary changes as instructed by the client.Upon completion of the cleaning phase, we proceeded with visualization.After visualization, we published the dashboard on Tableau Server and raised a ticket. Once the user approved, we updated the INC and closed it.

In the event that users required any changes to the visualization, we performed them directly on Tableau Server. This involved creating extracts and refreshing data. Additionally, we conducted operations such as sending visualizations via email at regular intervals.
